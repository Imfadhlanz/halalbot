## Cara Install Di Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> git clone https://github.com/alanxxYz/laanbot.git
> cd laanbot
> npm i
```

## Cara Install Di Windows
* [`Download Node JS`](https://nodejs.org/en/download/)
```cmd
> git clone https://github.com/alanxxYz/laanbot.git
> cd laanbot
> npm i
```

## Jalankan Bot
```cmd
> npm start
```
